## Golden Rectangle Cropping

### Common Aspect Ratios

| Format | Ratio | vs. Golden (1.618) |
|--------|-------|-------------------|
| 4:3 | 1.33 | Narrower |
| 3:2 | 1.5 | Slightly narrower |
| **Golden** | 1.618 | Perfect |
| 16:9 | 1.78 | Wider |
| 2:1 | 2.0 | Much wider |

### When to Use Golden Crop

- **Portraits**: Natural frame for head and shoulders
- **Product photography**: Elegant proportions
- **Fine art**: Classical aesthetic
- **Print**: Works well for traditional formats

## Practical Application

### In Lightroom/Photoshop

1. Enter Crop mode (R in Lightroom)
2. Press O to cycle overlays
3. Press Shift+O to rotate overlay
4. Available overlays:
   - Rule of Thirds
   - Golden Ratio
   - Golden Spiral
   - Triangle
   - Diagonal

### In-Camera

Some cameras offer golden ratio grid overlays:
- Check menu for "Grid display" options
- Third-party apps for smartphone cameras

### Mental Visualization

Quick approximation in the field:
- Thirds lines: 33% from edge
- Golden lines: ~38% from edge
- Place subject slightly inside the thirds intersection

## Genre-Specific Tips

### Street Photography
- Action following the spiral
- Eyes at intersection points
- Negative space in larger areas

### Architecture
- Building lines following diagonals
- Key features at golden points
- Symmetry can break golden rules (intentionally)

### Wildlife
- Eyes at spiral center
- Body following curve
- Direction of gaze into open space

### Macro/Product
- Key detail at focal point
- Surrounding elements follow curve
- Background in larger areas

---

## When NOT to Use Golden Ratio

### Centered Composition Works Better For:
- Symmetrical subjects
- Reflections
- Formal portraits
- Product shots requiring clarity

### Rule of Thirds Suffices For:
- Quick snapshots
- Documentary work
- Most casual photography

### Breaking All Rules:
- Avant-garde/artistic work
- Intentional tension
- Storytelling that requires discomfort

## Code: Calculate Golden Points

```python
def golden_points(width, height):
    """Calculate golden ratio intersection points"""
    phi = 1.618

    # Vertical lines (from edges)
    x1 = width / (1 + phi)         # ~38.2% from left
    x2 = width * phi / (1 + phi)   # ~61.8% from left

    # Horizontal lines (from edges)
    y1 = height / (1 + phi)        # ~38.2% from top
    y2 = height * phi / (1 + phi)  # ~61.8% from top

    # Intersection points
    points = [
        (x1, y1),  # Top-left
        (x2, y1),  # Top-right
        (x1, y2),  # Bottom-left
        (x2, y2),  # Bottom-right
    ]

    return points

# For a 1920x1080 image
points = golden_points(1920, 1080)
# [(733, 413), (1187, 413), (733, 667), (1187, 667)]
```

## Conclusion

The golden ratio in photography:

**Strengths:**
- Creates natural, flowing compositions
- Guides eye to focal point
- More dynamic than centered composition

**Limitations:**
- Not always appropriate
- Slightly harder to visualize than thirds
- Subject/story should drive composition

**Best practice:** Learn the rule, then know when to break it.
